# System Audit — 2026-07-06

Triggered by a real incident: an option position (ACN) sat at -68% loss for a week, uncaught, until manually closed. That led to a full audit of the live two-plane system (`execution_layer/alpha_plane.py`, `execution_layer/protection_plane.py`, and everything they call into) for the same two bug classes that caused it, plus anything else found along the way.

Three passes were run: (1) signature mismatches + gated exits + bracket-stop ordering, (2) IPC consistency between the two planes + breaker/wash-sale/alerting/broker math, (3) consensus orchestration internals + Kelly/correlation math + remaining alpha_plane.py surface + global shutdown wiring + scheduler cross-check.

**All 12 documented findings are now fixed or explicitly decided as of end of day 2026-07-06** (2 more real bugs — #16a, #16b — were discovered incidentally while testing #16's fix and fixed with the same rigor). Every fix has a regression test verified red-when-reverted; full suite (514 tests) green after all fixes combined.

**Post-fix re-audit (same day)**: a fourth audit pass specifically verified today's fixes themselves for correctness (not re-litigating the original bugs) — GlobalRiskState DB round-trip, the news-signal round-trip, the vol consensus graph fixes, the correlation fix's blast radius, and whether the new test files exercise real code paths rather than mocked-to-pass assertions. Clean across every section — no new issues found.

A fifth pass covered previously-unaudited surface (options_structurer.py, the remaining two vol sub-agents, config/settings.py validation, a fresh scheduler check, and a shadow-signals sanity pass) and found two more real issues, both fixed the same day:
